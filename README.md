# Typo Trap

정상 단어들 사이에 숨어있는 오타 단어를 3초 안에 찾아라!

---

## 게임 소개

여러 단어가 그리드 형태로 나열됩니다. 대부분은 정상 단어이고, 딱 하나만 오타가 있는 틀린 단어입니다.

오타 단어를 찾아 클릭하면 다음 단계로 넘어갑니다.  
정상 단어를 클릭하거나 3초가 지나면 게임 오버입니다.

## 플레이 방법

1. **시작 화면**에서 시작하기 버튼을 누릅니다.
2. **그리드**에 나열된 단어 중 오타가 있는 단어를 찾아 클릭합니다.
3. **정답**이면 다음 단계 버튼이 나타납니다.
4. **5단계**를 모두 클리어하면 게임이 끝납니다.

```
사자  사자
샤자  사자   ← '샤자'가 오타
```

## 단계별 난이도

단계가 올라갈수록 그리드가 커져서 오타를 찾기 어려워집니다.  
오타의 위치는 매 플레이마다 랜덤하게 바뀝니다.

| 단계 | 그리드 | 카드 수 |
|------|--------|---------|
| 1    | 2 × 2  | 4장     |
| 2    | 3 × 3  | 9장     |
| 3    | 4 × 4  | 16장    |
| 4    | 5 × 5  | 25장    |
| 5    | 5 × 5  | 25장    |

## 실행 방법

```bash
pnpm install
pnpm dev
```

브라우저에서 `http://localhost:5173` 으로 접속합니다.

```bash
pnpm build    # 프로덕션 빌드
pnpm preview  # 빌드 결과 미리보기
pnpm lint     # ESLint 검사
pnpm format   # Prettier 포맷
```

## 기술 스택

- [Pixi.js](https://pixijs.com/) v8 — 게임 렌더링
- [Vite](https://vitejs.dev/) — 번들러
- TypeScript
- ESLint / Prettier
